# Arduino-Assignment1-Smart Countdown
This is my submission for Assignment 1 in the
programmimg c++ for Engineers Arduino course
at[ Ghana communication Technology university computer egineering group A]
##What the program does
it counts down from 5 to 1 using a while loop.On each step:
.It prints "count: N" to the Serial Monitor.
.It blinks the on-board LED N Ttimes using a function called flashLED().
When the countdown finishes ,it prints "=== Countdown Complet ===".
##Concepts demonstrated 
-Variables (int)
-Functions with parameters (FlashLED)
-The While loop
-The While loop
-The Serial Monitor(Serial.begin,Serial.println)
##How to run it
1.Open the .ion file in the Aeduino IDE
2.Connect  and Arduino Uno via USB.
3.Select Tools>Board>Arduino Uno and the correct port.
4.Click upload ,then open Tools>SerialMonitor(9600 baud).
